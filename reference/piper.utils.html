piper.utils¶

piper.utils.dynamic_load(target)[source]¶

Dynamically import a class and return it

This is used by the core parts of the main configuration file since one of the main features is to let the user specify which class to use.

piper.utils.mkdir(path)[source]¶

Safe implementation of ‘mkdir -p’

piper.utils.now()[source]¶

Mockable now() method. datetime.datetime is immutable and cannot be patched by mock.

piper.utils.oneshot(cmd)[source]¶

Oneshot execution of a command. Returns output as a single string.

This is only to cut down on boiler.
